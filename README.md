# MurrayAbeles.com Replica

This repository provides a simple static copy of https://murrayabeles.com for
hosting on GitHub Pages or any static file host. It includes:

- `index.html` – home page with introduction and recent highlights
- `bio.html` – biography page
- `press.html` – press links and media appearances
- `contact.html` – contact form (static, non-functional)
- `css/style.css` – basic styling

To use it:

1. Push the contents of this folder to a GitHub repository.
2. Enable GitHub Pages from the repository settings (choose `main` branch).
3. Optionally, point your custom domain to GitHub Pages via DNS.

Images and logos remain linked from the original site (Squarespace CDN).

> **Note:** this is a minimal replication; you can expand styles and assets as needed.
